Expressões Aritméticas

Em algoritmo temos as expressões aritméticas onde temos divisão,multiplicação, porcentagem,adição,divisão inteira, resto da divisão e potenciação usamos dentro dos algoritmos e cada um deles tem uma cadeia de precedência primeiro o que esta dentro de () após potenciação,multiplicação, divisão,divisão inteira,porcentagem, adição e por fim subtração,vamos usar munto dentro dos futuros códigos.

Variáveis e Tipos de Dados

Variável é um espaço na memoria para guardar informação dentro do algoritmo, ela deve ser declarada dentro do programa onde declaramos o (nome):(tipo).Dentro da programação temos tipos de Variáveis tipo inteiro,real,logico e caractere.A variável o seu valor fica salvo sempre o ultimo valor informado, e temos que observar o nome dado para uma variável onde não pode ter espaço entre as palavras, não podemos usar as palavras reservadas ou caractere especiais algumas linguagens são Case-sensitive então devemos observar isso.

Operação Básicas da Programação

Temos a entrada de dados informados pelo usuário onde esses dados vão ficar armazenados nas variáveis já declaradas, após receber os dados temos o processamento de dados recebidos e por ultimo temos a saída de dados processados informados ao usuário através do monitor.

Cada linguagem tem alguma regra de entrada e saída de dados no caso do visualg o comando de saída de dados temos o ESCREVA("") E ESCREVAL(""), o ESCREVAL("") serve para pular uma linha usamos esse comando tanto para fazer uma pergunta ao usuário como para a saída do processamento dos dados, temos o comando LEIA que serve para ler o dados informado pelo usuário e guardar esses dados dentro das variáveis. E para guardar esses dados demos que já ter declarados as variáveis cada uma dentro do seu tipo corretamente, dentro do visualg para atribuir o valor a uma variável devemos usar <- em algumas linguagens usamos o =, esse comando estamos atribuindo um valor especifico a uma variável EX:
IDADE <- 25 (esse comando esta declarando que o valor de idade é 25).

No comando de saída para mostrar os valores da variáveis devemos declarar da seguinte forma
ESCREVA("a funcionaria", nome, "ganha", salario, "e tem ", idade)
Note que sempre devemos colocar uma virgula para fazer a separação das variáveis e as palavras que será imprimidas na tela do usuário.

Entrada de Dados

Entrada de dados é tudo o que o usuário informar para o programa através de um teclado e armazenando dentro da variável designada, assim o programa vai ler essa entrada assim dandFo continuidade ao programa, no visualg o comando para lar a entrada de dados é o LEIA(variável).

Estrutura Condicional

Expressões Comparativas

Dentro das estrutura condicionais temos os operadores comparativos, onde iremos usa-los muito para comparar valores onde podemos ter como resultado verdadeiro ou falso, com base em sua comparação exemplo 5 > 10 = falso.

Dentro os comparadores temos >(maior),(menor) <,(maior ou igual) >=,(menor ou igual) <=,(igual) =,(diferente) <>. Com eles vamos poder formar várias comparações dentro de um programa por isso ele retorna 2 valores verdadeiro ou falso.

Expressões Lógicas

São expressões lógicas que também retorna um valor verdadeiro ou falso mais usamos para fazer a comparação os Operadores Lógicos que dentro da programação são E, OU, Não.

Operador E = Conjunção será verdadeiro semente se ambas as expressões forem verdadeiras, operador OU = Disjunção será verdadeiro se pelos menos uma das dias condições forem verdadeiras, operador NÃO = Negação nega a condição caso ela seja verdadeira o valor que ira retornar é falso.

Estrutura Condicional

Ela é uma estrutura de controle onde um bloco de comando só será executado se a condição for verdadeira ou falsa com isso o programa irá executar o bloco referente a cada condição.

Temos a condição Simples conde temos uma unica condição onde só sera executado se uma unica condição for verdadeira caso a condição for falsa o programa o bloco de comando a sua sintaxe dentro do visualg é SE (condição) ENTAO (executa o bloco de comando) FIMSE.

Estrutura Composta temos 2 blocos de comando onde um sera executado caso a condição seja verdadeiro e outro bloco de comando caso a condição seja falsa e sua sintaxe é SE (condição) ENTÃO (bloco condição for verdadeiro) SENÃO (bloco de comando se condição for falso)FIMSE.

Estruturas Encadeadas ou Aninhadas podemos ter mais de 2 condição aninhando as estruturas encadeadas assim nosso código fica melhor na hora de se fazer alguma mudança dentro do código, sua sintaxe é SE (condição 1) ENTÃO (bloco comando 1) SENÃO SE (condição 2) ENTÃO (bloco comando 2) SENÃO (bloco comando 3)FIMSE, caso esteja com dificuldade de entender use o comando debug dentro do visualg é F8 assim você pode ver como o programa será executado.

Estrutura Escolha

Temos a estrutura escolha para facilitar algumas tomadas de decisão, usamos quando não temos que fazer comparativo de valor com expressões lógicas, podendo ser usado por exemplo na estrutura de um cardápio onde vai ter a descrição do prato e seu valor.

A sintaxe da estrutura escolha é ESCOLHA (variável) CASO 1 CASO 2 CASO 3 OUTROCASO FIMESCOLHA. A variável vai ser uma de controle onde sera determinante para o programa saber em qual caso vai jogar na tela para o usuário caso seja digitado um valor diferente de algum dos caso entra no bloco OUTROCASO que trara uma mensagem de valor invalido.

Estrutura Repetitivas

Enquanto

Essa é uma estrutura de controle que repete um bloco de comando enquanto uma condição for verdadeira, usamos essa estrutura quando não sabemos a quantidades de repetições que será realizada.

Sua sintaxe é ENQUANTO (condição)FACA (comandos)FIMENQUANDO
Regra:
Verdadeiro: executa os comandos e volta
Falso: pula fora do bloco de controle

Estrutura Repetitiva Para

É uma estrutura de controle que repete um bloco de comando até um certo intervalo de valores, muito usado quando sabemos quantas vezes um bloco determinado deve se repetir.

Sua sintaxe é PARA (variável) DE (valor_Inicial) ATE (valor_final)[passo_n]FACA(comandos) FIMPARA.

Dentro dessa estrutura temos uma variável usada para controlar o programa para saber quando o programa deve se encerrar, e vai se repetir de o valor inicial a qual voc~e vai colocar ate o numero de vezes que você também colocar já o passo_n serve para controlar se ela vai pular a contagem 2, 3, 4 ou posso diminuir fazendo uma contagem regressiva, e a regra diz que se nao for informado o programa por padrão vai ser passo 1.
