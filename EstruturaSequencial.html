Expressões Aritméticas

Em algoritmo temos as expressões aritméticas onde temos divisão,multiplicação, porcentagem,adição,divisão inteira, resto da divisão e potenciação usamos dentro dos algoritmos e cada um deles tem uma cadeia de precedência primeiro o que esta dentro de () após potenciação,multiplicação, divisão,divisão inteira,porcentagem, adição e por fim subtração,vamos usar munto dentro dos futuros códigos.

Variáveis e Tipos de Dados

Variável é um espaço na memoria para guardar informação dentro do algoritmo, ela deve ser declarada dentro do programa onde declaramos o (nome):(tipo).Dentro da programação temos tipos de Variáveis tipo inteiro,real,logico e caractere.A variável o seu valor fica salvo sempre o ultimo valor informado, e temos que observar o nome dado para uma variável onde não pode ter espaço entre as palavras, não podemos usar as palavras reservadas ou caractere especiais algumas linguagens são Case-sensitive então devemos observar isso.

Operação Básicas da Programação

Temos a entrada de dados informados pelo usuário onde esses dados vão ficar armazenados nas variáveis já declaradas, após receber os dados temos o processamento de dados recebidos e por ultimo temos a saída de dados processados informados ao usuário através do monitor.

Cada linguagem tem alguma regra de entrada e saída de dados no caso do visualg o comando de saída de dados temos o ESCREVA("") E ESCREVAL(""), o ESCREVAL("") serve para pular uma linha usamos esse comando tanto para fazer uma pergunta ao usuário como para a saída do processamento dos dados, temos o comando LEIA que serve para ler o dados informado pelo usuário e guardar esses dados dentro das variáveis. E para guardar esses dados demos que já ter declarados as variáveis cada uma dentro do seu tipo corretamente, dentro do visualg para atribuir o valor a uma variável devemos usar <- em algumas linguagens usamos o =, esse comando estamos atribuindo um valor especifico a uma variável EX:
IDADE <- 25 (esse comando esta declarando que o valor de idade é 25).

No comando de saída para mostrar os valores da variáveis devemos declarar da seguinte forma
ESCREVA("a funcionaria", nome, "ganha", salario, "e tem ", idade)
Note que sempre devemos colocar uma virgula para fazer a separação das variáveis e as palavras que será imprimidas na tela do usuário.
